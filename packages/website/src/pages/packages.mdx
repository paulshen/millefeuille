# Packages

millefeuille comes is available as two NPM packages, [`millefeuille`](https://www.npmjs.com/package/millefeuille)
and [`millefeuille-layer`](https://www.npmjs.com/package/millefeuille-layer). If
you only want the core Layer functionality, you can use `millefeuille-layer` to
save bytes.

## millefeuille-layer

This package contains the core `<Layer>` and `<LayerContainer>` components.

#### Exports

- `<Layer>`
- `<LayerContainer>`
- `LayerState`
- `APIMessageType`
- `LayerAPI`
- `createAPI`

## millefeuille

This includes everything in `millefeuille-layer` as well as a number of common
UI components.

#### Exports

- `<PopperLayer>`
- `<Tooltip>`, `TooltipConfigContext`
- `<Dialog>`, `<DialogLayer>`, `DialogConfigContext`
