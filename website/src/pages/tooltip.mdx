import { Tooltip } from "millefeuille"

# Tooltip

<Tooltip text="Tooltip Text">
  {tooltipProps => <button {...tooltipProps}>Hover Me</button>}
</Tooltip>

Tooltip is a component built on top of PopperLayer that manages mouse interactions. Tooltip has to attach mouse listeners and a ref to the context element. This is done with an explicit children render API.

<!-- prettier-ignore -->
```js
import { Tooltip } from "millefeuille"

<Tooltip text="Tooltip Text">
  {tooltipProps => <button {...tooltipProps}>Hover Me</button>}
</Tooltip>
```

## Tooltip API

| Prop                                  | Type                          | Required |
| ------------------------------------- | ----------------------------- | -------- |
| [`children`](#children)               | `(TooltipProps) => ReactNode` | true     |
| [`text`](#text)                       | `ReactNode`                   | true     |
| [`placement`](#placement)             | `Placement`                   | false    |
| [`tooltipRenderer`](#tooltiprenderer) | `Function`                    | false    |

### children

###### Type

<!-- prettier-ignore -->
```typescript
(tooltipProps: {
  onMouseEnter: () => void;
  onMouseLeave: () => void;
  ref: React.RefObject<any>;
}) => ReactNode
```

A render function for the tooltip contents. Tooltip adopts a children render
prop API to give you control how to render the children. You are responsible for
transferring the props to your rendered element. This makes it more explicit,
for example, that your child needs to handle `onMouseEnter` and `onMouseLeave`.

### text

###### Type

```typescript
ReactNode
```

The contents of the tooltip

### placement

###### Optional Type

<pre class="language-typescript">
  <a href="https://github.com/popperjs/popper-core/blob/1c380a42cccc5286786bcd73872f2f5157a99db8/src/enums.js#L41">
    Placement
  </a>
</pre>

Popper.js Placement value where to render the tooltip (e.g. `"bottom"`,
`"left"`, ..)

### tooltipRenderer

###### Optional Type

<pre class="language-typescript">
  {`(
  text: ReactNode,
  props: { popperState: `}
  <a href="https://github.com/popperjs/popper-core/blob/1c380a42cccc5286786bcd73872f2f5157a99db8/src/types.js#L54-L77">
    State
  </a>
  {` | undefined }
) => ReactNode`}
</pre>

A custom renderer for the tooltip. You can use this to render an arrow, etc.

<!-- prettier-ignore -->
```typescript
function customTooltipRender(text: string, { popperState }) {
  return <div>{text}</div>
}

<Tooltip text="Text" tooltipRenderer={customTooltipRender}>
  {tooltipProps => <button {...tooltipProps}>Child</button>}
</Tooltip>
```
