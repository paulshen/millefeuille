# Tooltip

<Tooltip text="Tooltip Text">
  {tooltipProps => <button {...tooltipProps}>Hover Me</button>}
</Tooltip>

Tooltip is a component built on top of PopperLayer that manages mouse
interactions. Tooltip attaches mouse listeners and a ref to the context element.
This is implemented with an explicit children render API.

**Be sure to transfer the provided props to the child element!**

<!-- prettier-ignore -->
```typescript{4}
import { Tooltip } from "millefeuille"

<Tooltip text="Tooltip Text">
  {tooltipProps => <button {...tooltipProps}>Hover Me</button>}
</Tooltip>
```

## Tooltip Render Context

You will almost certainly want to customize the render tooltip function. This is
done via React Context.

###### Type

<pre class="language-typescript">
  {`(text: ReactNode, props: { popperState: `}
  <a href="https://github.com/popperjs/popper-core/blob/1c380a42cccc5286786bcd73872f2f5157a99db8/src/types.js#L54-L77">
    State
  </a>
  {` | undefined }) => ReactNode`}
</pre>

Using the provided Popper.js [`State`](https://github.com/popperjs/popper-core/blob/1c380a42cccc5286786bcd73872f2f5157a99db8/src/types.js#L54-L77),
you can get the placement of the tooltip and more. This is useful if you are
rendering tooltip arrows, for example.

<!-- prettier-ignore -->
```typescript
import { TooltipReactContext } from "millefeuille"

function renderTooltip(text, popperState) {
  return <div>{text}</div>
}

function MyApp() {
  return (
    <TooltipReactContext.Provider value={renderTooltip}>
      ...
    </TooltipReactContext.Provider>
  )
}
```

## Tooltip API

| Prop                      | Type                          | Required |
| ------------------------- | ----------------------------- | -------- |
| [`children`](#children)   | `(TooltipProps) => ReactNode` | true     |
| [`text`](#text)           | `ReactNode`                   | true     |
| [`placement`](#placement) | `Placement`                   | false    |

### children

###### Type

<!-- prettier-ignore -->
```typescript
(tooltipProps: {
  onMouseEnter: () => void;
  onMouseLeave: () => void;
  ref: React.RefObject<any>;
}) => ReactNode
```

A render function for the tooltip contents. Tooltip adopts a children render
prop API to give you control how to render the children. You are responsible for
transferring the props to your rendered element. This makes it more explicit,
for example, that your child needs to handle `onMouseEnter` and `onMouseLeave`.

### text

###### Type

```typescript
ReactNode
```

The contents of the tooltip, usually a string but can be any React element.

### placement

###### Optional Type

<pre class="language-typescript">
  <a href="https://github.com/popperjs/popper-core/blob/1c380a42cccc5286786bcd73872f2f5157a99db8/src/enums.js#L41">
    Placement
  </a>
</pre>

Popper.js Placement value where to render the tooltip (e.g. `"bottom"`,
`"left"`, ..). The default value is `"top"`.
